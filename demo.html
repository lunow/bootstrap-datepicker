Datepicker with Selected Option

This modified version of the bootstrap datepicker accept an array of objects with dates and tooltips, highlighted in the datepicker.

Demo 
$('.datepicker').datepicker({
	selected: [
		{
			date: '2012-09-12'
		},
		{ 
			date: '2012-09-03',
			tooltip: '1 Entry'
		},
		{ 
			date: '2012-09-17',
			tooltip: '4 Entries'
		},
		{ 
			date: '2012-09-01',
			tooltip: '3 Entries'
		},
		{ 
			date: '2012-08-30',
			tooltip: '1 Entry'
		},
		{ 
			date: '2012-10-15',
			tooltip: '1 Entry'
		}
	]
});
